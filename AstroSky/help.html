1. Quick Start — First Image in 60 Seconds

You can go from zero to a beautiful exported image in under a minute. Here's how:

  1. 1

    Install & Launch

    Download AstroSky from the Microsoft Store and launch it. No account or sign-up required.

  2. 2

    Import Your FITS Files

    Click Import Files on the Gallery toolbar, or simply drag and drop FITS files or a folder directly onto the application window.

  3. 3

    Click a Thumbnail

    Click any thumbnail in the Gallery grid. The image renders instantly on the canvas below, and the Parameter Panel on the right activates.

  4. 4

    Hit Auto

    Click the Auto Enhance button in the Parameter Panel. AstroSky analyzes the histogram and automatically picks the best stretch algorithm, color map, and black/white points.

  5. 5

    Export

    Click Export Image at the bottom of the Parameter Panel. Choose a resolution preset (1080p, 1440p, 4K, 5K, or 8K), pick a format (PNG or JPEG), and click Export. Your wallpaper-ready image is saved.

Pro Tip: You can batch-import an entire folder of FITS files at once. AstroSky loads them in parallel and shows real-time progress for each file.

2. Importing FITS Files

Supported File Types

AstroSky supports .fits, .fit, and .fts files. It handles both single-extension FITS and multi-extension FITS (MEF) files, as well as data cubes where NAXIS=3 (e.g., radio spectral-line cubes).

How to Import

There are three ways to load files into AstroSky:

  • Import Files button — Click the Import Files button on the Gallery toolbar to open a file picker dialog.
  • Drag and drop files — Drag one or more FITS files from Explorer directly onto the Gallery area.
  • Drag and drop folders — Drag an entire folder onto the Gallery area. AstroSky recursively finds all FITS files inside.

Import Progress

Each file loads with a real-time progress indicator. Files are processed in parallel using multiple CPU cores. You can cancel all pending imports at any time by clicking the Cancel All button on the toolbar.

Note: Large data cubes (hundreds of spectral channels) may take a few seconds to load. AstroSky remains responsive during import — you can continue working with already-loaded images.

3. Auto Smart Enhance

The Auto button is the fastest way to make any FITS image look great. It's the core of AstroSky's "click once, looks amazing" philosophy.

What Auto Does

When you click Auto, AstroSky:

  1. Reads the full pixel value histogram of the current image.
  2. Classifies the histogram shape — unimodal (single peak), bimodal (two peaks), or long-tail (extended faint-end distribution).
  3. Selects the optimal stretch algorithm for that histogram shape (e.g., Asinh for long-tail, Linear for bimodal).
  4. Picks the most appropriate color map (Viridis for general-purpose, Inferno for high-contrast, Hubble Palette for emission-line data).
  5. Sets black point and white point clipping values to maximize visible detail.
  6. Renders the result in under 200ms.
Tip: Auto is a great starting point. After clicking Auto, you can still manually adjust the stretch, color map, and black/white points to fine-tune the result.

4. Stretch Algorithms & Color Maps

Stretch Algorithms

The stretch algorithm determines how raw FITS pixel values are mapped to display brightness. AstroSky offers:

  • Linear — Direct proportional mapping. Best for images with balanced dynamic range.
  • Asinh — Hyperbolic arcsine stretch (similar to Lupton et al. 2004). Excellent for preserving both bright and faint features — the default for most use cases.
  • Sqrt — Square-root stretch. Enhances faint structure while compressing bright peaks.
  • Log — Logarithmic stretch. Maximum contrast in faint regions; good for data with extreme dynamic range.
  • Power (Gamma) — Power-law mapping with adjustable gamma exponent.

Live Histogram

The histogram panel shows a real-time 256-bin histogram of your image. Two draggable markers indicate the black point (everything below is clipped to black) and white point (everything above is clipped to white). Drag either marker and the preview updates in 200ms — no waiting.

Color Maps

Choose from a curated set of perceptually uniform, colorblind-friendly color maps:

  • Viridis — General-purpose, perceptually uniform (default).
  • Plasma — High-contrast warm-to-cool gradient.
  • Inferno — Black-body-like; excellent for high-dynamic-range data.
  • Magma — Earth-tone variant with strong luminance contrast.
  • Hubble Palette — Inspired by the iconic Hubble Space Telescope color scheme.
  • Grayscale — Pure luminance; ideal for scientific analysis.
  • Heat — Classic thermal map.

Each color map displays a gradient preview swatch in the dropdown menu, so you can preview before selecting.

5. Beauty Mode vs. Science Mode

AstroSky has two interface modes, toggled by the switch in the top Chrome toolbar. The switch takes less than 100ms and does not reload your images.

Beauty Mode (Default)

Designed for a clean, creative experience:

  • Parameter Panel shows only Auto, Color Map, and Export.
  • No scientific overlays — just the rendered image.
  • Ideal for creating wallpapers and shareable images.

Science Mode

Expands the full analysis toolkit for astronomers:

  • Beam Ellipse Overlay — Visualizes telescope beam from BMAJ/BMIN FITS headers.
  • Pixel Value Readout — Hover over any pixel to see its value in the status bar.
  • Sky Info Overlay — Displays celestial coordinates on the image.
  • Color Bar — Shows the intensity-to-color mapping scale.
  • PV Diagrams — Position-velocity slice extraction for data cubes.
  • Channel Maps — Full channel map grid view for spectral cubes.
  • Moment Maps — Compute moment-0 (integrated intensity) and moment-1 (velocity field).
  • Image Algebra — Add, subtract, multiply, or divide two images.

6. RGB Channel Synthesis

Combine three separate FITS images (e.g., different wavelength bands) into a single full-color composite.

How to Synthesize

  1. 1

    Import Images

    Load at least three FITS images into the Gallery (e.g., R-band, G-band, B-band data).

  2. 2

    Assign Channels

    Hover over any thumbnail to reveal R / G / B channel buttons. Click to assign that image to a channel. Alternatively, right-click a thumbnail and assign via context menu.

  3. 3

    Synthesize

    Once all three channels are assigned, click the RGB Synthesis (3/3) button on the Gallery toolbar. The synthesis dialog lets you adjust per-channel stretch and weight.

Tip: The RGB Synthesis button shows your channel assignment progress in real-time — e.g., (1/3), (2/3), (3/3). It highlights when all three channels are filled.

7. Science Tools

AstroSky includes a suite of analysis tools for professional astronomers, all available in Science Mode.

FITS Header Viewer

Accessible from the Header tab in the Parameter Panel. Shows all FITS header entries in a searchable three-column table (Keyword / Value / Comment). Click any value to copy it. Use the extension switcher to browse headers from multi-extension files.

Celestial Object Database

Accessible from the Info tab. When a FITS file contains WCS (World Coordinate System) headers, AstroSky automatically:

  • Reads the celestial coordinates (RA/Dec in Equatorial, l/b in Galactic).
  • Cross-matches against a built-in offline database of 800+ celestial objects (Messier, NGC, IC catalogs).
  • Displays plain-language descriptions of any matched objects.
  • Shows an Aitoff-projection sky minimap with your field-of-view marked.

Beam Ellipse Overlay

If your FITS headers contain BMAJ, BMIN, and BPA keywords (beam major axis, minor axis, position angle), AstroSky overlays a semi-transparent beam ellipse on the image canvas. Toggle visibility from the canvas toolbar.

Moment Maps

For data cubes (NAXIS=3 FITS files), AstroSky can compute moment maps:
Moment 0 — Integrated intensity (sum along the spectral axis).
Moment 1 — Intensity-weighted mean velocity field.
Access via the Gallery toolbar's Moment Maps button.

Position-Velocity (PV) Diagrams

For data cubes, define a slice line on the image canvas and AstroSky extracts the position-velocity diagram along that slice. Access via the canvas toolbar in Science Mode.

Image Algebra

Perform arithmetic operations between two images: Addition, Subtraction, Multiplication, Division. Access via the Gallery toolbar's Image Algebra button.

8. Exporting Images

Single Image Export

  1. 1

    Open Export Dialog

    With an image selected, click the Export Image button at the bottom of the Parameter Panel.

  2. 2

    Choose Format

    Select PNG (lossless, best quality), JPEG (smaller file size), or FITS (scientific data preservation).

  3. 3

    Choose Resolution

    Pick a preset: 1920x1080, 2560x1440, 3840x2160 (4K), 5120x2880 (5K), or 7680x4320 (8K). AstroSky renders at the target resolution regardless of screen size.

  4. 4

    Add Watermark (Optional)

    Optionally embed a watermark with telescope model, observation band, integration time, and author signature — perfect for sharing on AstroBin or CloudyNights.

Batch Export

Click Export All on the Gallery toolbar to export all loaded images in one operation. Select individual images with Ctrl+Click, then use Export Selected from the context menu.

12. Frequently Asked Questions

What FITS file formats does AstroSky support?

AstroSky supports .fits, .fit, and .fts files — both single-extension and multi-extension FITS (MEF). Data cubes with NAXIS=3 are fully supported with spectral channel navigation.

Can I use AstroSky completely offline?

Yes. After installation from the Microsoft Store, AstroSky works 100% offline. All FITS loading, image processing, and rendering runs locally on your device. No internet connection is needed.

Is my data sent anywhere?

No. AstroSky processes all data in memory on your local machine. No files, images, or any data are ever uploaded to the cloud or any third party. There is no telemetry or analytics.

Does AstroSky work on Mac or Linux?

Currently, AstroSky is a Windows application (Windows 10 and Windows 11). It is built on .NET 10 + WPF and distributed through the Microsoft Store. Mac and Linux versions are not available at this time.

What is the maximum image size AstroSky can handle?

AstroSky is optimized for large FITS files from modern surveys. Images up to 16K × 16K pixels are processed efficiently. Performance depends on your system's RAM and CPU cores — 8 GB RAM is recommended for 4K+ images.

Can I export images at different aspect ratios?

Yes. In the Export dialog, you can choose between Fit to Canvas (preserves the original FITS image aspect ratio) or Crop to Fill (fills the chosen wallpaper resolution by cropping the image edges).

Does Auto Enhance work on all types of FITS data?

Auto Enhance is designed for general-purpose continuum and line-emission data (the most common astronomy images). For specialized data like spectral cubes or polarization maps, manual adjustment after Auto is recommended.

How do I update AstroSky?

Updates are delivered automatically through the Microsoft Store. Open the Microsoft Store, go to Library, and click Get Updates to check for the latest version.

Where can I get sample FITS files to try?

Several free public archives offer FITS data: ESO Science Archive, NRAO VLA Archive, NASA/IPAC Infrared Science Archive, and Herschel Science Archive. Download any calibrated FITS image and drop it into AstroSky.

Can I use AstroSky for professional research?

Yes. AstroSky's Science Mode provides analysis tools suitable for professional radio astronomy and survey data inspection, including beam overlays, pixel readout, PV diagrams, and moment map computation. However, for advanced photometry or spectral analysis, dedicated tools like DS9 or CASA are recommended.

13. Troubleshooting

Image not rendering / blank canvas

  • Ensure the FITS file contains image data (check NAXIS ≥ 2 in the header).
  • Try clicking Auto — it resets all stretch parameters and recalculates the optimal display window.
  • Check that black/white point sliders are not both at extreme values.

Import fails or file is rejected

  • Verify the file extension is .fits, .fit, or .fts.
  • Ensure the file is not corrupted — try opening it in another FITS viewer like DS9.
  • If importing a large folder, try importing individual files to identify problematic ones.

Performance is slow

  • Close other GPU-intensive applications (AstroSky uses SkiaSharp GPU rendering).
  • For very large data cubes, consider extracting individual channels before loading.
  • Ensure your graphics drivers are up to date.

WCS / Sky Coordinates not showing

  • The FITS file must contain standard WCS header keywords (CRVAL1/2, CRPIX1/2, CDELT1/2, CTYPE1/2, etc.).
  • If your file lacks WCS headers, the Sky Info and Celestial Database features will not be available for that image.
