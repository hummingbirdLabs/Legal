1. Avvio Rapido — Prima Immagine in 60 Secondi

Puoi passare da zero a una splendida immagine esportata in meno di un minuto. Ecco come:

  1. 1

    Installa e Avvia

    Scarica AstroSky dal Microsoft Store e avvialo. Nessun account o registrazione richiesti.

  2. 2

    Importa i Tuoi File FITS

    Clicca Importa File sulla barra degli strumenti della Galleria, o semplicemente trascina e rilascia file FITS o una cartella direttamente nella finestra dell'applicazione.

  3. 3

    Clicca una Miniatura

    Clicca su qualsiasi miniatura nella griglia della Galleria. L'immagine viene renderizzata istantaneamente sul canvas sottostante e il Pannello dei Parametri a destra si attiva.

  4. 4

    Premi Auto

    Clicca il pulsante Auto Enhance nel Pannello dei Parametri. AstroSky analizza l'istogramma e sceglie automaticamente il miglior algoritmo di stretch, la mappa di colori e i punti nero/bianco.

  5. 5

    Esporta

    Clicca Esporta Immagine in fondo al Pannello dei Parametri. Scegli una risoluzione predefinita (1080p, 1440p, 4K, 5K o 8K), seleziona un formato (PNG o JPEG) e clicca Esporta. La tua immagine pronta per sfondo viene salvata.

Consiglio da Pro: Puoi importare in batch un'intera cartella di file FITS in una sola volta. AstroSky li carica in parallelo e mostra l'avanzamento in tempo reale per ogni file.

2. Importazione di File FITS

Tipi di File Supportati

AstroSky supporta file .fits, .fit e .fts. Gestisce sia file FITS a estensione singola che multi-estensione (MEF), nonché cubi di dati dove NAXIS=3 (es. cubi di righe spettrali radio).

Come Importare

Ci sono tre modi per caricare file in AstroSky:

  • Pulsante Importa File — Clicca il pulsante Importa File sulla barra degli strumenti della Galleria per aprire il selettore file.
  • Trascina e rilascia file — Trascina uno o più file FITS da Explorer direttamente nell'area Galleria.
  • Trascina e rilascia cartelle — Trascina un'intera cartella nell'area Galleria. AstroSky trova ricorsivamente tutti i file FITS al suo interno.

Avanzamento dell'Importazione

Ogni file viene caricato con un indicatore di avanzamento in tempo reale. I file vengono elaborati in parallelo utilizzando più core della CPU. Puoi annullare tutte le importazioni in sospeso in qualsiasi momento cliccando il pulsante Annulla Tutto sulla barra degli strumenti.

Nota: I cubi di dati grandi (centinaia di canali spettrali) potrebbero richiedere alcuni secondi per il caricamento. AstroSky rimane reattivo durante l'importazione — puoi continuare a lavorare con le immagini già caricate.

3. Auto Enhance Intelligente

Il pulsante Auto è il modo più veloce per rendere splendida qualsiasi immagine FITS. È il nucleo della filosofia AstroSky "un clic, aspetto straordinario".

Cosa Fa Auto

Quando clicchi Auto, AstroSky:

  1. Legge l'istogramma completo dei valori dei pixel dell'immagine corrente.
  2. Classifica la forma dell'istogramma — unimodale (un solo picco), bimodale (due picchi) o coda lunga (distribuzione estesa dei deboli).
  3. Seleziona l'algoritmo di stretch ottimale per quella forma di istogramma (es. Asinh per coda lunga, Lineare per bimodale).
  4. Sceglie la mappa di colori più appropriata (Viridis per uso generico, Inferno per alto contrasto, Palette Hubble per dati di righe di emissione).
  5. Imposta i valori di clipping del punto nero e punto bianco per massimizzare i dettagli visibili.
  6. Renderizza il risultato in meno di 200ms.
Consiglio: Auto è un ottimo punto di partenza. Dopo aver cliccato Auto, puoi comunque regolare manualmente lo stretch, la mappa di colori e i punti nero/bianco per perfezionare il risultato.

4. Algoritmi di Stretch e Mappe di Colori

Algoritmi di Stretch

L'algoritmo di stretch determina come i valori grezzi dei pixel FITS vengono mappati alla luminosità del display. AstroSky offre:

  • Lineare — Mappatura proporzionale diretta. Ideale per immagini con gamma dinamica bilanciata.
  • Asinh — Stretch iperbolico con arcoseno (simile a Lupton et al. 2004). Eccellente per preservare sia strutture brillanti che deboli — predefinito per la maggior parte dei casi.
  • Sqrt — Stretch a radice quadrata. Esalta le strutture deboli comprimendo i picchi brillanti.
  • Log — Stretch logaritmico. Massimo contrasto nelle regioni deboli; ideale per dati con gamma dinamica estrema.
  • Power (Gamma) — Mappatura a legge di potenza con esponente gamma regolabile.

Istogramma in Tempo Reale

Il pannello dell'istogramma mostra un istogramma a 256 bin in tempo reale della tua immagine. Due marcatori trascinabili indicano il punto nero (tutto ciò che è sotto viene tagliato a nero) e il punto bianco (tutto ciò che è sopra viene tagliato a bianco). Trascina qualsiasi marcatore e l'anteprima si aggiorna in 200ms — nessuna attesa.

Mappe di Colori

Scegli da un set curato di mappe di colori percettivamente uniformi e adatte ai daltonici:

  • Viridis — Uso generico, percettivamente uniforme (predefinita).
  • Plasma — Gradiente ad alto contrasto da caldo a freddo.
  • Inferno — Simile al corpo nero; eccellente per dati ad alta gamma dinamica.
  • Magma — Variante toni terra con forte contrasto di luminanza.
  • Palette Hubble — Ispirata all'iconico schema di colori del Telescopio Spaziale Hubble.
  • Scala di Grigi — Luminanza pura; ideale per analisi scientifiche.
  • Heat — Mappa termica classica.

Ogni mappa di colori mostra un campione del gradiente nel menu a discesa, così puoi vedere l'anteprima prima di selezionare.

5. Modalità Bellezza vs. Modalità Scienza

AstroSky ha due modalità di interfaccia, commutabili tramite l'interruttore nella barra Chrome superiore. Il cambio richiede meno di 100ms e non ricarica le tue immagini.

Modalità Bellezza (Predefinita)

Progettata per un'esperienza pulita e creativa:

  • Il Pannello dei Parametri mostra solo Auto, Mappa Colori ed Esporta.
  • Nessuna sovrapposizione scientifica — solo l'immagine renderizzata.
  • Ideale per creare sfondi e immagini condivisibili.

Modalità Scienza

Espande il toolkit completo di analisi per astronomi:

  • Sovrapposizione Ellisse del Fascio — Visualizza il fascio del telescopio dagli header FITS BMAJ/BMIN.
  • Lettura Valore Pixel — Passa il mouse su qualsiasi pixel per vederne il valore nella barra di stato.
  • Sovrapposizione Info Celesti — Mostra le coordinate celesti sull'immagine.
  • Barra dei Colori — Mostra la scala di mappatura intensità-colore.
  • Diagrammi PV — Estrazione di sezioni posizione-velocità per cubi di dati.
  • Mappe dei Canali — Vista a griglia completa delle mappe dei canali per cubi spettrali.
  • Mappe dei Momenti — Calcola momento-0 (intensità integrata) e momento-1 (campo di velocità).
  • Algebra di Immagini — Somma, sottrai, moltiplica o dividi due immagini.

6. Sintesi Canali RGB

Combina tre immagini FITS separate (es. diverse bande di lunghezza d'onda) in un'unica composizione a colori.

Come Sintetizzare

  1. 1

    Importa Immagini

    Carica almeno tre immagini FITS nella Galleria (es. dati bande R, G, B).

  2. 2

    Assegna Canali

    Passa il mouse su qualsiasi miniatura per rivelare i pulsanti canale R / G / B. Clicca per assegnare quell'immagine a un canale. In alternativa, clicca col tasto destro su una miniatura e assegna dal menu contestuale.

  3. 3

    Sintetizza

    Una volta assegnati tutti e tre i canali, clicca il pulsante Sintesi RGB (3/3) sulla barra degli strumenti della Galleria. Il dialogo di sintesi permette di regolare stretch e peso per canale.

Consiglio: Il pulsante Sintesi RGB mostra l'avanzamento dell'assegnazione dei canali in tempo reale — es. (1/3), (2/3), (3/3). Si evidenzia quando tutti e tre i canali sono pieni.

7. Strumenti Scientifici

AstroSky include una suite di strumenti di analisi per astronomi professionisti, tutti disponibili in Modalità Scienza.

Visualizzatore Header FITS

Accessibile dalla scheda Header nel Pannello dei Parametri. Mostra tutte le voci dell'header FITS in una tabella ricercabile a tre colonne (Parola Chiave / Valore / Commento). Clicca su qualsiasi valore per copiarlo. Usa il selettore di estensione per navigare tra gli header di file multi-estensione.

Database Oggetti Celesti

Accessibile dalla scheda Info. Quando un file FITS contiene header WCS (World Coordinate System), AstroSky automaticamente:

  • Legge le coordinate celesti (RA/Dec in Equatoriale, l/b in Galattica).
  • Effettua il cross-match con un database offline integrato di oltre 800 oggetti celesti (cataloghi Messier, NGC, IC).
  • Mostra descrizioni in linguaggio semplice di qualsiasi oggetto corrispondente.
  • Visualizza una minimappa celeste in proiezione Aitoff con il tuo campo visivo marcato.

Sovrapposizione Ellisse del Fascio

Se i tuoi header FITS contengono le parole chiave BMAJ, BMIN e BPA (asse maggiore del fascio, asse minore, angolo di posizione), AstroSky sovrappone un'ellisse del fascio semitrasparente sul canvas dell'immagine. Attiva/disattiva la visibilità dalla barra degli strumenti del canvas.

Mappe dei Momenti

Per cubi di dati (file FITS con NAXIS=3), AstroSky può calcolare mappe dei momenti:
Momento 0 — Intensità integrata (somma lungo l'asse spettrale).
Momento 1 — Campo di velocità medio pesato per intensità.
Accesso tramite il pulsante Mappe dei Momenti sulla barra degli strumenti della Galleria.

Diagrammi Posizione-Velocità (PV)

Per cubi di dati, definisci una linea di sezione sul canvas dell'immagine e AstroSky estrae il diagramma posizione-velocità lungo quella sezione. Accesso dalla barra degli strumenti del canvas in Modalità Scienza.

Algebra di Immagini

Esegui operazioni aritmetiche tra due immagini: Addizione, Sottrazione, Moltiplicazione, Divisione. Accesso dal pulsante Algebra di Immagini sulla barra degli strumenti della Galleria.

8. Esportazione di Immagini

Esportazione Immagine Singola

  1. 1

    Apri Dialogo di Esportazione

    Con un'immagine selezionata, clicca il pulsante Esporta Immagine in fondo al Pannello dei Parametri.

  2. 2

    Scegli Formato

    Seleziona PNG (senza perdita, qualità migliore), JPEG (dimensioni file ridotte) o FITS (conservazione dati scientifici).

  3. 3

    Scegli Risoluzione

    Scegli un predefinito: 1920x1080, 2560x1440, 3840x2160 (4K), 5120x2880 (5K) o 7680x4320 (8K). AstroSky renderizza alla risoluzione target indipendentemente dalle dimensioni dello schermo.

  4. 4

    Aggiungi Filigrana (Opzionale)

    Opzionalmente, incorpora una filigrana con modello del telescopio, banda di osservazione, tempo di integrazione e firma dell'autore — perfetto per condividere su AstroBin o CloudyNights.

Esportazione in Batch

Clicca Esporta Tutto sulla barra degli strumenti della Galleria per esportare tutte le immagini caricate in un'unica operazione. Seleziona immagini individuali con Ctrl+Clic, quindi usa Esporta Selezionate dal menu contestuale.

12. Domande Frequenti

Quali formati di file FITS supporta AstroSky?

AstroSky supporta file .fits, .fit e .fts — sia FITS a estensione singola che multi-estensione (MEF). I cubi di dati con NAXIS=3 sono completamente supportati con navigazione dei canali spettrali.

Posso usare AstroSky completamente offline?

Sì. Dopo l'installazione dal Microsoft Store, AstroSky funziona al 100% offline. Tutto il caricamento FITS, l'elaborazione delle immagini e la renderizzazione vengono eseguiti localmente sul tuo dispositivo. Nessuna connessione internet è necessaria.

I miei dati vengono inviati da qualche parte?

No. AstroSky elabora tutti i dati in memoria sulla tua macchina locale. Nessun file, immagine o dato viene mai caricato sul cloud o su terze parti. Non c'è telemetria o analisi.

AstroSky funziona su Mac o Linux?

Attualmente, AstroSky è un'applicazione Windows (Windows 10 e Windows 11). È sviluppato su .NET 10 + WPF e distribuito tramite Microsoft Store. Le versioni per Mac e Linux non sono al momento disponibili.

Qual è la dimensione massima dell'immagine che AstroSky può gestire?

AstroSky è ottimizzato per grandi file FITS da survey moderne. Immagini fino a 16K × 16K pixel vengono elaborate in modo efficiente. Le prestazioni dipendono dalla RAM e dai core della CPU del tuo sistema — 8 GB di RAM sono consigliati per immagini 4K+.

Posso esportare immagini con proporzioni diverse?

Sì. Nel dialogo Esporta, puoi scegliere tra Adatta al Canvas (preserva le proporzioni originali dell'immagine FITS) o Ritaglia per Riempire (riempie la risoluzione sfondo scelta ritagliando i bordi dell'immagine).

Auto Enhance funziona su tutti i tipi di dati FITS?

Auto Enhance è progettato per dati di continuo e righe di emissione di uso generale (le immagini astronomiche più comuni). Per dati specializzati come cubi spettrali o mappe di polarizzazione, si raccomanda la regolazione manuale dopo Auto.

Come si aggiorna AstroSky?

Gli aggiornamenti vengono distribuiti automaticamente tramite Microsoft Store. Apri Microsoft Store, vai su Libreria e clicca Verifica Aggiornamenti per controllare l'ultima versione.

Dove posso trovare file FITS di esempio da provare?

Diversi archivi pubblici gratuiti offrono dati FITS: ESO Science Archive, NRAO VLA Archive, NASA/IPAC Infrared Science Archive e Herschel Science Archive. Scarica qualsiasi immagine FITS calibrata e trascinala in AstroSky.

Posso usare AstroSky per la ricerca professionale?

Sì. La Modalità Scienza di AstroSky fornisce strumenti di analisi adatti alla radioastronomia professionale e all'ispezione di dati survey, inclusi sovrapposizioni del fascio, lettura pixel, diagrammi PV e calcolo mappe dei momenti. Tuttavia, per fotometria avanzata o analisi spettrale, si raccomandano strumenti dedicati come DS9 o CASA.

13. Risoluzione dei Problemi

Immagine non visualizzata / canvas vuoto

  • Assicurati che il file FITS contenga dati immagine (controlla NAXIS ≥ 2 nell'header).
  • Prova a cliccare Auto — reimposta tutti i parametri di stretch e ricalcola la finestra di visualizzazione ottimale.
  • Controlla che i cursori dei punti nero/bianco non siano entrambi a valori estremi.

Importazione fallita o file rifiutato

  • Verifica che l'estensione del file sia .fits, .fit o .fts.
  • Assicurati che il file non sia corrotto — prova ad aprirlo in un altro visualizzatore FITS come DS9.
  • Se stai importando una cartella grande, prova a importare file singoli per identificare quelli problematici.

Prestazioni lente

  • Chiudi altre applicazioni che usano intensivamente la GPU (AstroSky usa il rendering GPU SkiaSharp).
  • Per cubi di dati molto grandi, considera di estrarre canali individuali prima del caricamento.
  • Assicurati che i driver grafici siano aggiornati.

WCS / Coordinate Celesti non visualizzate

  • Il file FITS deve contenere parole chiave header WCS standard (CRVAL1/2, CRPIX1/2, CDELT1/2, CTYPE1/2, ecc.).
  • Se il tuo file manca di header WCS, le funzionalità Info Celesti e Database Celeste non saranno disponibili per quell'immagine.
Ancora problemi? I file di log diagnostici vengono scritti nella directory dell'applicazione e cancellati a ogni avvio. Puoi inviare questi log a HummingbirdLabs@outlook.com per assistenza. Nessun dato personale è incluso nei file di log.
