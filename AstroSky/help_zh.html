1. 快速入门 — 60 秒内创建第一张图像

一分钟内即可获得精美的导出图像。操作步骤如下：

  1. 1

    安装并启动

    从 Microsoft Store 下载 AstroSky 并启动。无需账号或注册。

  2. 2

    导入 FITS 文件

    在图库工具栏中点击导入文件，或将 FITS 文件或文件夹直接拖放到应用程序窗口中。

  3. 3

    点击缩略图

    在图库网格中点击缩略图。图像将立即渲染到画布，右侧参数面板随之激活。

  4. 4

    点击自动

    在参数面板中点击自动增强按钮。AstroSky 将分析直方图并自动选择最佳的拉伸算法、色彩映射和黑/白点。

  5. 5

    导出

    点击参数面板底部的导出图像。选择分辨率预设（1080p、1440p、4K、5K、8K），选择格式（PNG 或 JPEG），然后点击导出。您的壁纸级图像就保存好了。

小技巧：您可以将包含 FITS 文件的整个文件夹一次性批量导入。AstroSky 会并行加载文件，并显示每个文件的实时进度。

2. 导入 FITS 文件

支持的文件类型

AstroSky 支持 .fits、.fit、.fts 文件。支持单扩展 FITS 和多扩展 FITS (MEF) 文件，以及 NAXIS=3 数据立方体（如射电谱线立方体）。

导入方式

有三种方式将文件加载到 AstroSky：

  • 导入文件按钮 — 在图库工具栏中点击导入文件按钮，打开文件选择对话框。
  • 拖放文件 — 从资源管理器将一个或多个 FITS 文件直接拖入图库区域。
  • 拖放文件夹 — 将整个文件夹拖入图库区域。AstroSky 会递归查找其中的所有 FITS 文件。

导入进度

每个文件以实时进度指示器加载。文件使用多个 CPU 核心并行处理。您可以通过工具栏中的全部取消按钮随时取消所有待处理的导入。

注意：大数据立方体（数百个光谱通道）可能需要几秒钟加载。在导入过程中，AstroSky 保持响应，您可以继续处理已加载的图像。

3. 自动智能增强

自动按钮是让任何 FITS 图像看起来惊艳的最快方式。它体现了 AstroSky "一键即美"的核心理念。

自动按钮的工作原理

当您点击自动时，AstroSky 会：

  1. 读取当前图像的完整像素值直方图。
  2. 对直方图形状进行分类 — 单峰（单一峰值）、双峰（两个峰值）或长尾（扩展的微弱信号分布）。
  3. 选择最适合该直方图形状的拉伸算法（例如，长尾用 Asinh，双峰用 Linear）。
  4. 选择最合适的色彩映射（通用 Viridis、高对比度 Inferno、或发射线数据专用的 Hubble Palette）。
  5. 设置黑点和白点裁剪值以最大化可见细节。
  6. 在 200ms 内渲染结果。
提示：自动是一个很好的起点。点击自动后，您仍然可以手动调整拉伸、色彩映射和黑/白点来微调结果。

4. 拉伸算法与色彩映射

拉伸算法

拉伸算法决定原始 FITS 像素值如何映射到显示亮度。AstroSky 提供：

  • Linear — 直接比例映射。最适合动态范围均衡的图像。
  • Asinh — 双曲反正弦拉伸（类似 Lupton et al. 2004）。在保留亮暗特征方面表现出色，大多数情况下的默认选择。
  • Sqrt — 平方根拉伸。增强微弱结构的同时压缩亮峰。
  • Log — 对数拉伸。在暗弱区域提供最大对比度，适合极端动态范围的图像。
  • Power (Gamma) — 可调 gamma 指数的幂律映射。

实时直方图

直方图面板显示图像的实时 256-bin 直方图。两个可拖动的标记代表黑点（以下裁剪为黑色）和白点（以上裁剪为白色）。拖动标记时，预览在 200ms 内即时更新。

色彩映射

从精心挑选的感知均匀且色盲友好的色彩映射中选择：

  • Viridis — 通用，感知均匀（默认）。
  • Plasma — 高对比度冷暖渐变。
  • Inferno — 类黑体，适合高动态范围数据。
  • Magma — 大地色调变体，具有强烈的亮度对比。
  • Hubble Palette — 灵感来自标志性的哈勃太空望远镜配色。
  • Grayscale — 纯亮度，科学分析理想选择。
  • Heat — 经典热力图。

每个色彩映射在下拉菜单中显示渐变预览色块，方便您在选择前预览。

5. 美学模式 vs 科学模式

AstroSky 有两种界面模式，通过顶部 Chrome 工具栏的开关切换。切换仅需 <100ms，不会重新加载图像。

美学模式（默认）

为简洁、创意的体验而设计：

  • 参数面板仅显示自动、色彩映射和导出。
  • 画布仅显示渲染后的图像，无科学叠加层。
  • 非常适合创建壁纸和可分享的图像。

科学模式

扩展完整的天文分析工具集：

  • 波束椭圆叠加 — 从 BMAJ/BMIN FITS 头部关键字可视化望远镜波束。
  • 像素值读取 — 鼠标悬停在像素上时，状态栏显示其值。
  • 天球坐标叠加 — 在图像上显示天球坐标。
  • 色阶 — 显示强度-颜色映射比例尺。
  • PV 图 — 从数据立方体中提取位置-速度切片。
  • 通道图 — 光谱立方体的全通道图网格视图。
  • 矩图 — 矩-0（积分强度）和矩-1（速度场）计算。
  • 图像运算 — 两幅图像的加、减、乘、除。

6. RGB 通道合成

将三幅独立的 FITS 图像（例如，不同波段的观测数据）合并为一幅全彩合成图像。

合成方法

  1. 1

    导入图像

    至少将三幅 FITS 图像加载到图库中（例如 R 波段、G 波段、B 波段数据）。

  2. 2

    分配通道

    鼠标悬停在缩略图上时，会出现 R / G / B 通道按钮。点击将图像分配到对应通道。也可以右键点击缩略图通过上下文菜单分配。

  3. 3

    合成

    三个通道都分配完毕后，点击图库工具栏中的 RGB 合成 (3/3) 按钮。在合成对话框中可以调整每个通道的拉伸和权重。

提示：RGB 合成按钮实时显示通道分配进度，如 (1/3)、(2/3)、(3/3)。三个通道就绪时会高亮显示。

7. 科学工具

AstroSky 包含一套面向专业天文学家的分析工具，全部在科学模式下可用。

FITS 头部查看器

通过参数面板的头部选项卡访问。以可搜索的三列表格显示所有 FITS 头部条目（关键字 / 值 / 描述）。点击值即可复制。通过扩展切换器浏览多扩展文件的头部。

天体数据库

通过信息选项卡访问。如果 FITS 文件包含 WCS（世界坐标系）头部，AstroSky 会自动：

  • 读取天球坐标（赤道坐标系中的 RA/Dec，银河坐标系中的 l/b）。
  • 与内置离线数据库中的800+ 个天体（Messier、NGC、IC 星表）进行交叉匹配。
  • 显示匹配天体的通俗语言描述。
  • 显示带有视场标记的 Aitoff 投影天球迷你地图。

波束椭圆叠加

如果 FITS 头部包含 BMAJ、BMIN、BPA 关键字（波束长轴、短轴、位置角），AstroSky 会在图像画布上叠加半透明波束椭圆。通过画布工具栏切换可见性。

矩图

对于数据立方体（NAXIS=3 FITS 文件），AstroSky 可以计算矩图：
矩 0 — 积分强度（沿光谱轴求和）。
矩 1 — 强度加权平均速度场。
通过图库工具栏中的矩图按钮访问。

位置-速度 (PV) 图

对于数据立方体，在图像画布上定义切片线，AstroSky 会提取沿该切片的位置-速度图。通过科学模式下的画布工具栏访问。

图像运算

对两幅图像进行算术运算：加、减、乘、除。通过图库工具栏中的图像运算按钮访问。

8. 导出图像

单张图像导出

  1. 1

    打开导出对话框

    选中图像后，点击参数面板底部的导出图像按钮。

  2. 2

    选择格式

    选择 PNG（无损，最高质量）、JPEG（文件更小）或 FITS（保留科学数据）。

  3. 3

    选择分辨率

    从预设中选择：1920x1080、2560x1440、3840x2160 (4K)、5120x2880 (5K) 或 7680x4320 (8K)。无论屏幕尺寸如何，AstroSky 都会以目标分辨率渲染。

  4. 4

    添加水印（可选）

    可以选择插入包含望远镜型号、观测波段、积分时间和作者署名的文字水印 — 非常适合在 AstroBin 或 CloudyNights 上分享。

批量导出

使用图库工具栏中的全部导出可一次性导出所有已加载的图像。通过 Ctrl+点击选择单个图像，然后使用上下文菜单中的导出选中。

12. 常见问题

AstroSky 支持哪些 FITS 文件格式？

AstroSky 支持 .fits、.fit 和 .fts 文件 — 包括单扩展和多扩展 FITS (MEF)。NAXIS=3 数据立方体完全支持，可进行光谱通道浏览。

可以完全离线使用 AstroSky 吗？

可以。从 Microsoft Store 安装后，AstroSky 100% 离线运行。所有 FITS 加载、图像处理和渲染均在您的设备上本地运行。无需互联网连接。

我的数据会被上传到任何地方吗？

不会。AstroSky 在您本地机器的内存中处理所有数据。没有任何文件、图像或数据被上传到云端或第三方。没有遥测或分析。

AstroSky 在 Mac 或 Linux 上能用吗？

目前 AstroSky 是 Windows 应用程序（Windows 10 和 Windows 11）。基于 .NET 10 + WPF 构建，通过 Microsoft Store 分发。暂无 Mac 和 Linux 版本。

AstroSky 能处理多大的图像？

AstroSky 针对现代巡天的大文件 FITS 进行了优化。最高可高效处理 16K×16K 像素的图像。性能取决于系统 RAM 和 CPU 核心 — 4K+ 图像建议 8GB RAM。

可以用不同宽高比导出图像吗？

可以。在导出对话框中，您可以选择适应画布（保持原始 FITS 图像的宽高比）或裁剪填充（裁剪图像边缘以填满所选壁纸分辨率）。

自动增强对每种 FITS 数据都有效吗？

自动增强专为一般连续谱和线发射数据（最常见的天文图像类型）设计。对于像光谱立方体或偏振图这样的特殊数据，建议在自动之后进行手动调整。

如何更新 AstroSky？

更新通过 Microsoft Store 自动提供。打开 Microsoft Store，进入库，点击获取更新以检查最新版本。

哪里可以获取测试用的样例 FITS 文件？

多个免费公共档案提供 FITS 数据：ESO Science Archive、NRAO VLA Archive、NASA/IPAC Infrared Science Archive、Herschel Science Archive。下载校准后的 FITS 图像，拖入 AstroSky 即可。

AstroSky 能用于专业研究吗？

能。AstroSky 的科学模式提供适合专业射电天文学和巡天数据检查的分析工具，包括波束叠加、像素读取、PV 图和矩图计算。但对于高级测光或光谱分析，建议使用 DS9 或 CASA 等专用工具。

13. 故障排除

图像不渲染 / 画布为空

  • 确保 FITS 文件包含图像数据（检查 NAXIS ≥ 2 in 头部）。
  • 尝试点击自动 — 这将重置所有拉伸参数并重新计算最佳显示范围。
  • 检查黑/白点滑块是否未处于极端位置。

导入失败或文件被拒绝

  • 确保文件扩展名为 .fits、.fit 或 .fts。
  • 确认文件未损坏 — 尝试在另一个 FITS 查看器中打开（如 DS9）。
  • 批量导入文件夹时，尝试单独导入文件以识别问题文件。

性能缓慢

  • 关闭其他 GPU 密集型应用程序（AstroSky 使用 SkiaSharp GPU 渲染）。
  • 对于非常大的数据立方体，考虑在加载前提取单个通道。
  • 确保您的显卡驱动是最新的。

WCS / 天球坐标不显示

  • FITS 文件必须包含标准 WCS 头部关键字（CRVAL1/2、CRPIX1/2、CDELT1/2、CTYPE1/2 等）。
  • 如果文件没有 WCS 头部，则该图像无法使用天体信息和天体数据库功能。
