1. 빠른 시작 — 60초 안에 첫 이미지 만들기

1분 이내에 아름다운 내보내기 이미지를 얻을 수 있습니다. 방법은 다음과 같습니다:

  1. 1

    설치 및 실행

    Microsoft Store에서 AstroSky를 다운로드하고 실행하세요. 계정이나 가입이 필요하지 않습니다.

  2. 2

    FITS 파일 가져오기

    갤러리 도구 모음에서 파일 가져오기를 클릭하거나 FITS 파일이나 폴더를 애플리케이션 창에 직접 드래그 앤 드롭하세요.

  3. 3

    썸네일 클릭

    갤러리 그리드에서 썸네일을 클릭하세요. 이미지가 캔버스에 즉시 렌더링되고 오른쪽 매개변수 패널이 활성화됩니다.

  4. 4

    자동 버튼 클릭

    매개변수 패널에서 자동 보정 버튼을 클릭하세요. AstroSky가 히스토그램을 분석하고 최적의 스트레치 알고리즘, 색상 맵, 흑백 포인트를 자동으로 선택합니다.

  5. 5

    내보내기

    매개변수 패널 하단의 이미지 내보내기를 클릭하세요. 해상도 사전 설정(1080p, 1440p, 4K, 5K, 8K)을 선택하고 형식(PNG 또는 JPEG)을 선택한 후 내보내기를 클릭하세요. 배경화면용 이미지가 저장됩니다.

프로 팁: FITS 파일이 들어 있는 전체 폴더를 한 번에 일괄 가져올 수 있습니다. AstroSky는 파일을 병렬로 로드하며 각 파일의 실시간 진행 상황을 표시합니다.

2. FITS 파일 가져오기

지원 파일 유형

AstroSky는 .fits, .fit, .fts 파일을 지원합니다. 단일 확장 FITS 및 다중 확장 FITS(MEF) 파일과 NAXIS=3 데이터 큐브(예: 전파 스펙트럼 선 큐브)를 모두 처리합니다.

가져오기 방법

AstroSky에 파일을 로드하는 세 가지 방법이 있습니다:

  • 파일 가져오기 버튼 — 갤러리 도구 모음에서 파일 가져오기 버튼을 클릭하여 파일 선택 대화상자를 엽니다.
  • 파일 드래그 앤 드롭 — 탐색기에서 하나 이상의 FITS 파일을 갤러리 영역으로 직접 드래그하세요.
  • 폴더 드래그 앤 드롭 — 전체 폴더를 갤러리 영역으로 드래그하세요. AstroSky가 내부의 모든 FITS 파일을 재귀적으로 찾습니다.

가져오기 진행 상황

각 파일은 실시간 진행 표시기와 함께 로드됩니다. 파일은 여러 CPU 코어를 사용하여 병렬로 처리됩니다. 도구 모음의 모두 취소 버튼을 클릭하여 언제든지 보류 중인 모든 가져오기를 취소할 수 있습니다.

참고: 대용량 데이터 큐브(수백 개의 스펙트럼 채널)는 로드하는 데 몇 초가 걸릴 수 있습니다. 가져오기 중에도 AstroSky는 응답성을 유지하며 이미 로드된 이미지로 작업을 계속할 수 있습니다.

3. 자동 스마트 보정

자동 버튼은 모든 FITS 이미지를 멋지게 만드는 가장 빠른 방법입니다. AstroSky의 "한 번 클릭하면 놀랍게 보인다" 철학의 핵심입니다.

자동 버튼의 작동 방식

자동을 클릭하면 AstroSky는:

  1. 현재 이미지의 전체 픽셀 값 히스토그램을 읽습니다.
  2. 히스토그램 모양을 분류합니다 — 단봉(단일 피크), 이중봉(두 개의 피크), 또는 긴 꼬리(확장된 약한 신호 분포).
  3. 해당 히스토그램 모양에 최적의 스트레치 알고리즘을 선택합니다(예: 긴 꼬리에는 Asinh, 이중봉에는 Linear).
  4. 가장 적절한 색상 맵을 선택합니다(범용 Viridis, 고대비 Inferno, 방출선 데이터용 Hubble Palette).
  5. 가시적인 세부 사항을 최대화하기 위해 흑점과 백점 클리핑 값을 설정합니다.
  6. 200ms 이내에 결과를 렌더링합니다.
팁: 자동은 훌륭한 시작점입니다. 자동을 클릭한 후에도 스트레치, 색상 맵, 흑백 포인트를 수동으로 조정하여 결과를 미세 조정할 수 있습니다.

자동 보정을 직접 사용해 보시겠습니까?

AstroSky를 무료로 다운로드하세요 — 30일 체험, 모든 기능 잠금 해제, 계정 불필요.

4. 스트레치 알고리즘 및 색상 맵

스트레치 알고리즘

스트레치 알고리즘은 원시 FITS 픽셀 값이 표시 밝기로 매핑되는 방식을 결정합니다. AstroSky는 다음을 제공합니다:

  • Linear — 직접 비례 매핑. 균형 잡힌 동적 범위의 이미지에 가장 적합합니다.
  • Asinh — 쌍곡선 아크사인 스트레치(Lupton et al. 2004와 유사). 밝은 특징과 희미한 특징을 모두 보존하는 데 탁월하며 대부분의 경우 기본값입니다.
  • Sqrt — 제곱근 스트레치. 밝은 피크를 압축하면서 희미한 구조를 강화합니다.
  • Log — 로그 스트레치. 희미한 영역에서 최대 대비를 제공하며 극단적인 동적 범위의 데이터에 적합합니다.
  • Power (Gamma) — 조정 가능한 감마 지수를 사용한 거듭제곱 법칙 매핑.

실시간 히스토그램

히스토그램 패널은 이미지의 실시간 256-빈 히스토그램을 표시합니다. 두 개의 드래그 가능한 마커가 흑점(아래는 검정색으로 클리핑)과 백점(위는 흰색으로 클리핑)을 나타냅니다. 마커를 드래그하면 미리보기가 200ms 이내에 업데이트됩니다.

색상 맵

지각적으로 균일하고 색각 이상 친화적인 큐레이션된 색상 맵 세트에서 선택하세요:

  • Viridis — 범용, 지각적으로 균일(기본값).
  • Plasma — 고대비 웜-쿨 그라데이션.
  • Inferno — 흑체 유사, 높은 동적 범위 데이터에 탁월.
  • Magma — 강한 휘도 대비를 가진 어스톤 변형.
  • Hubble Palette — 상징적인 허블 우주 망원경 색 구성표에서 영감을 받음.
  • Grayscale — 순수 휘도, 과학적 분석에 이상적.
  • Heat — 클래식 열 지도.

각 색상 맵은 드롭다운 메뉴에 그라데이션 미리보기 견본을 표시하므로 선택하기 전에 미리 볼 수 있습니다.

5. 뷰티 모드 vs 사이언스 모드

AstroSky에는 두 가지 인터페이스 모드가 있으며, 상단 Chrome 도구 모음의 스위치로 전환할 수 있습니다. 전환은 100ms 미만이 소요되며 이미지를 다시 로드하지 않습니다.

뷰티 모드 (기본값)

깨끗하고 창의적인 경험을 위해 설계되었습니다:

  • 매개변수 패널에 자동, 색상 맵, 내보내기만 표시됩니다.
  • 과학적 오버레이 없이 렌더링된 이미지만 표시됩니다.
  • 배경화면 및 공유 가능한 이미지를 만드는 데 이상적입니다.

사이언스 모드

천문학자를 위한 전체 분석 도구 키트를 확장합니다:

  • 빔 타원 오버레이 — BMAJ/BMIN FITS 헤더에서 망원경 빔을 시각화합니다.
  • 픽셀 값 읽기 — 픽셀 위에 마우스를 올리면 상태 표시줄에 값이 표시됩니다.
  • 천체 좌표 오버레이 — 이미지에 천체 좌표를 표시합니다.
  • 색상 막대 — 강도-색상 매핑 스케일을 표시합니다.
  • PV 다이어그램 — 데이터 큐브에 대한 위치-속도 슬라이스 추출.
  • 채널 맵 — 스펙트럼 큐브에 대한 전체 채널 맵 그리드 보기.
  • 모멘트 맵 — 모멘트-0(통합 강도) 및 모멘트-1(속도장) 계산.
  • 이미지 연산 — 두 이미지의 덧셈, 뺄셈, 곱셈, 나눗셈.

6. RGB 채널 합성

세 개의 개별 FITS 이미지(예: 서로 다른 파장 대역)를 하나의 풀컬러 합성 이미지로 결합합니다.

합성 방법

  1. 1

    이미지 가져오기

    최소 세 개의 FITS 이미지를 갤러리에 로드하세요(예: R-band, G-band, B-band 데이터).

  2. 2

    채널 할당

    썸네일 위에 마우스를 올리면 R / G / B 채널 버튼이 나타납니다. 클릭하여 해당 이미지를 채널에 할당하세요. 또는 썸네일을 마우스 오른쪽 버튼으로 클릭하고 컨텍스트 메뉴를 통해 할당할 수 있습니다.

  3. 3

    합성

    세 채널이 모두 할당되면 갤러리 도구 모음에서 RGB 합성 (3/3) 버튼을 클릭하세요. 합성 대화상자에서 채널별 스트레치와 가중치를 조정할 수 있습니다.

팁: RGB 합성 버튼은 채널 할당 진행 상황을 실시간으로 표시합니다 — 예: (1/3), (2/3), (3/3). 세 채널이 모두 채워지면 강조 표시됩니다.

7. 과학 도구

AstroSky에는 전문 천문학자를 위한 분석 도구 모음이 포함되어 있으며, 모두 사이언스 모드에서 사용할 수 있습니다.

FITS 헤더 뷰어

매개변수 패널의 헤더 탭에서 접근할 수 있습니다. 모든 FITS 헤더 항목을 검색 가능한 3열 테이블(키워드 / 값 / 설명)로 표시합니다. 값을 클릭하여 복사할 수 있습니다. 확장 전환기를 사용하여 다중 확장 파일의 헤더를 탐색할 수 있습니다.

천체 데이터베이스

정보 탭에서 접근할 수 있습니다. FITS 파일에 WCS(세계 좌표계) 헤더가 포함된 경우 AstroSky는 자동으로:

  • 천체 좌표(적도 좌표계의 RA/Dec, 은하 좌표계의 l/b)를 읽습니다.
  • 내장 오프라인 데이터베이스의 800개 이상의 천체(Messier, NGC, IC 카탈로그)와 교차 매칭합니다.
  • 일치하는 천체의 평이한 언어 설명을 표시합니다.
  • 시야가 표시된 Aitoff 투영 천구 미니맵을 표시합니다.

빔 타원 오버레이

FITS 헤더에 BMAJ, BMIN, BPA 키워드(빔 장축, 단축, 위치각)가 포함된 경우 AstroSky는 이미지 캔버스에 반투명 빔 타원을 오버레이합니다. 캔버스 도구 모음에서 가시성을 전환할 수 있습니다.

모멘트 맵

데이터 큐브(NAXIS=3 FITS 파일)의 경우 AstroSky는 모멘트 맵을 계산할 수 있습니다:
모멘트 0 — 통합 강도(스펙트럼 축을 따라 합산).
모멘트 1 — 강도 가중 평균 속도장.
갤러리 도구 모음의 모멘트 맵 버튼을 통해 접근할 수 있습니다.

위치-속도(PV) 다이어그램

데이터 큐브의 경우 이미지 캔버스에 슬라이스 선을 정의하면 AstroSky가 해당 슬라이스를 따라 위치-속도 다이어그램을 추출합니다. 사이언스 모드의 캔버스 도구 모음을 통해 접근할 수 있습니다.

이미지 연산

두 이미지 간의 산술 연산을 수행합니다: 덧셈, 뺄셈, 곱셈, 나눗셈. 갤러리 도구 모음의 이미지 연산 버튼을 통해 접근할 수 있습니다.

8. 이미지 내보내기

단일 이미지 내보내기

  1. 1

    내보내기 대화상자 열기

    이미지를 선택한 상태에서 매개변수 패널 하단의 이미지 내보내기 버튼을 클릭하세요.

  2. 2

    형식 선택

    PNG(무손실, 최고 품질), JPEG(더 작은 파일 크기), 또는 FITS(과학 데이터 보존)를 선택하세요.

  3. 3

    해상도 선택

    사전 설정을 선택하세요: 1920x1080, 2560x1440, 3840x2160 (4K), 5120x2880 (5K), 또는 7680x4320 (8K). AstroSky는 화면 크기에 관계없이 대상 해상도로 렌더링합니다.

  4. 4

    워터마크 추가 (선택 사항)

    선택적으로 망원경 모델, 관측 대역, 통합 시간 및 작성자 서명이 포함된 워터마크를 삽입할 수 있습니다 — AstroBin 또는 CloudyNights에서 공유하기에 완벽합니다.

일괄 내보내기

갤러리 도구 모음에서 모두 내보내기를 클릭하여 로드된 모든 이미지를 한 번의 작업으로 내보낼 수 있습니다. Ctrl+클릭으로 개별 이미지를 선택한 다음 컨텍스트 메뉴에서 선택한 항목 내보내기를 사용하세요.

9. 설정 및 언어

설정 열기

오른쪽 상단 Chrome 도구 모음의 톱니바퀴 아이콘을 클릭하여 설정 창을 엽니다.

테마

라이트와 다크 테마 중에서 선택하세요. 테마는 다시 시작하지 않고 즉시 적용됩니다.

언어

AstroSky는 7개 언어로 완전히 로컬라이즈되어 있습니다:

  • English (기본값)
  • 日本語 (Japanese)
  • Deutsch (German)
  • Français (French)
  • Español (Spanish)

12. 자주 묻는 질문

AstroSky는 어떤 FITS 파일 형식을 지원하나요?

AstroSky는 .fits, .fit, .fts 파일을 지원하며 — 단일 확장 및 다중 확장 FITS(MEF) 모두를 지원합니다. NAXIS=3 데이터 큐브는 스펙트럼 채널 탐색과 함께 완전히 지원됩니다.

완전히 오프라인으로 AstroSky를 사용할 수 있나요?

네. Microsoft Store에서 설치한 후 AstroSky는 100% 오프라인으로 작동합니다. 모든 FITS 로딩, 이미지 처리 및 렌더링은 기기에서 로컬로 실행됩니다. 인터넷 연결이 필요하지 않습니다.

제 데이터가 어딘가로 전송되나요?

아니요. AstroSky는 모든 데이터를 로컬 머신의 메모리에서 처리합니다. 파일, 이미지 또는 어떤 데이터도 클라우드나 제3자에게 업로드되지 않습니다. 텔레메트리나 분석도 없습니다.

AstroSky가 Mac이나 Linux에서 작동하나요?

현재 AstroSky는 Windows 애플리케이션입니다(Windows 10 및 Windows 11). .NET 10 + WPF로 구축되었으며 Microsoft Store를 통해 배포됩니다. 현재 Mac 및 Linux 버전은 제공되지 않습니다.

AstroSky가 처리할 수 있는 최대 이미지 크기는 얼마인가요?

AstroSky는 현대 관측의 대용량 FITS 파일에 최적화되어 있습니다. 최대 16K × 16K 픽셀 이미지가 효율적으로 처리됩니다. 성능은 시스템 RAM과 CPU 코어에 따라 달라집니다 — 4K+ 이미지에는 8GB RAM이 권장됩니다.

다양한 화면 비율로 이미지를 내보낼 수 있나요?

네. 내보내기 대화상자에서 캔버스에 맞춤(원본 FITS 이미지 화면 비율 유지) 또는 채우기 위해 자르기(이미지 가장자리를 잘라 선택한 배경화면 해상도에 채움) 중에서 선택할 수 있습니다.

자동 보정이 모든 유형의 FITS 데이터에 작동하나요?

자동 보정은 범용 연속체 및 선 방출 데이터(가장 일반적인 천문 이미지)를 위해 설계되었습니다. 스펙트럼 큐브나 편광 맵과 같은 특수 데이터의 경우 자동 후 수동 조정이 권장됩니다.

AstroSky를 어떻게 업데이트하나요?

업데이트는 Microsoft Store를 통해 자동으로 제공됩니다. Microsoft Store를 열고 라이브러리로 이동하여 업데이트 확인을 클릭하면 최신 버전을 확인할 수 있습니다.

테스트할 샘플 FITS 파일은 어디서 구할 수 있나요?

여러 무료 공개 아카이브에서 FITS 데이터를 제공합니다: ESO Science Archive, NRAO VLA Archive, NASA/IPAC Infrared Science Archive, Herschel Science Archive. 보정된 FITS 이미지를 다운로드하여 AstroSky에 드롭하세요.

전문 연구에 AstroSky를 사용할 수 있나요?

네. AstroSky의 사이언스 모드는 빔 오버레이, 픽셀 읽기, PV 다이어그램, 모멘트 맵 계산 등 전문 전파 천문학 및 관측 데이터 검사에 적합한 분석 도구를 제공합니다. 그러나 고급 측광 또는 스펙트럼 분석에는 DS9 또는 CASA와 같은 전용 도구가 권장됩니다.

13. 문제 해결

이미지가 렌더링되지 않음 / 빈 캔버스

  • FITS 파일에 이미지 데이터가 포함되어 있는지 확인하세요(헤더에서 NAXIS ≥ 2 확인).
  • 자동을 클릭해 보세요 — 모든 스트레치 매개변수를 재설정하고 최적의 표시 창을 다시 계산합니다.
  • 흑백 포인트 슬라이더가 둘 다 극단값에 있지 않은지 확인하세요.

가져오기 실패 또는 파일이 거부됨

  • 파일 확장자가 .fits, .fit 또는 .fts인지 확인하세요.
  • 파일이 손상되지 않았는지 확인하세요 — DS9와 같은 다른 FITS 뷰어에서 열어보세요.
  • 대용량 폴더를 가져오는 경우 개별 파일을 가져와 문제가 있는 파일을 식별해 보세요.

성능이 느림

  • 다른 GPU 집약적 애플리케이션을 닫으세요(AstroSky는 SkiaSharp GPU 렌더링을 사용합니다).
  • 매우 큰 데이터 큐브의 경우 로드하기 전에 개별 채널을 추출하는 것이 좋습니다.
  • 그래픽 드라이버가 최신 상태인지 확인하세요.

WCS / 천체 좌표가 표시되지 않음

  • FITS 파일에 표준 WCS 헤더 키워드(CRVAL1/2, CRPIX1/2, CDELT1/2, CTYPE1/2 등)가 포함되어 있어야 합니다.
  • 파일에 WCS 헤더가 없으면 해당 이미지에 대해 천체 정보 및 천체 데이터베이스 기능을 사용할 수 없습니다.
여전히 문제가 있나요? 진단 로그 파일은 애플리케이션 디렉터리에 기록되며 시작할 때마다 지워집니다. 이 로그를 HummingbirdLabs@outlook.com으로 보내 지원을 받을 수 있습니다. 로그 파일에는 개인 데이터가 포함되지 않습니다.
