Windows용 전문 FITS 이미지 시각화 도구. 천문 관측 데이터를 가져와 지능형 스트레치 알고리즘과 컬러맵을 적용하고, 바탕화면 품질의 이미지를 몇 초 만에 내보낼 수 있습니다. 완전 오프라인.

주요 기능

FITS 파일에서 아름다운 이미지로

천문학의 Snapseed — 간단하고 즉각적인 피드백, 시각 중심. 복잡한 매개변수는 고급 옵션에 숨기고, 바로 사용할 수 있는 경험을 제공합니다.

📂

FITS 일괄 가져오기

여러 FITS 파일이나 전체 폴더를 드래그 앤 드롭. 실시간 진행률과 함께 병렬 로딩. .fits, .fit, .fts 확장자 및 다중 확장 FITS 파일 지원.

✨

스마트 자동 보정

원클릭 Auto 버튼이 히스토그램 형태(단봉/이중봉/긴꼬리)를 분석하고 최적의 스트레치 알고리즘, 컬러맵, 흑백점 조합을 자동 선택합니다.

🎨

풍부한 컬러맵

Viridis, Plasma, Inferno, Magma, Hubble Palette 등에서 선택. 각 컬러맵은 그라데이션 미리보기를 포함합니다. 모두 지각적으로 균일하고 색각 이상 친화적.

📊

실시간 히스토그램

256개 빈의 실시간 히스토그램과 흑백점 마커. 슬라이더를 움직이면 200ms 즉시 미리보기 업데이트 — 기다림이 없습니다.

🔭

뷰티 / 사이언스 듀얼 모드

뷰티 모드는 과학적 노이즈를 숨겨 깨끗한 창의적 경험을 제공합니다. 사이언스 모드는 빔 오버레이, 픽셀 판독, PV 다이어그램, 채널 맵 등 전체 분석 도구를 확장 — ≤100ms 내 전환, 재로딩 불필요.

🌌

RGB 채널 합성

썸네일 위 호버 버튼으로 R/G/B 채널을 서로 다른 FITS 이미지에 할당. 채널별 스트레치와 가중치 제어로 다중 대역 합성 이미지 생성.

🖼️

바탕화면 품질 내보내기

1080p에서 8K까지 사전 설정 해상도로 PNG/JPEG/FITS 내보내기. 망원경 모델, 관측 대역, 적분 시간이 포함된 선택적 워터마크. 저자 서명 지원.

🗺️

천체 정보 및 천체 데이터베이스

WCS 헤더에서 좌표(은하/적도) 자동 인식. 800개 이상의 천체(Messier, NGC, IC)가 포함된 내장 오프라인 데이터베이스와 쉬운 설명. Aitoff 투영 천구 미니맵.

📋

FITS 헤더 뷰어

모든 FITS 헤더 항목에 대한 검색 가능한 3열 테이블(키워드/값/코멘트). 원클릭 값 복사. 확장 전환기를 통한 다중 확장 지원.

📐

과학 도구

모멘트 맵, 이미지 대수학, 위치-속도 다이어그램, 데이터 큐브(NAXIS=3)용 채널 맵. BMAJ/BMIN 헤더 키워드의 빔 타원 오버레이. 호버 시 픽셀 값 판독.

↩️

실행 취소 / 다시 실행

Ctrl+Z / Ctrl+Y를 통한 전체 매개변수 실행 취소/다시 실행 스택. 모든 조정이 추적되고 되돌릴 수 있습니다. 비파괴 처리 — 원본 FITS 데이터는 절대 변경되지 않습니다.

🌐

10개 언어 현지화

영어, 일본어, 독일어, 프랑스어, 스페인어, 포르투갈어, 러시아어, 한국어, 이탈리아어, 아랍어의 전체 UI 현지화. 설정에서 언제든지 재시작 없이 언어 전환 가능.

개인정보 보호 우선

데이터는 기기를 떠나지 않습니다

모든 이미지 처리는 로컬 컴퓨터에서 완전히 실행됩니다. 계정 필요 없음. 설치 후 인터넷 연결 불필요.

🔒

100% 오프라인

모든 FITS 로딩, 스트레치 처리, 렌더링이 로컬에서 실행됩니다. Microsoft Store에서 설치 후 완전히 오프라인으로 작동합니다.

🚫

데이터 업로드 제로

FITS 파일과 렌더링된 이미지는 사용자 컴퓨터의 메모리에서 처리됩니다. 원격 측정, 분석, 어떤 종류의 데이터 전송도 없습니다.

👤

계정 불필요

가입, 로그인, 구독이 필요 없습니다. Microsoft Store에서 다운로드하고 즉시 천문 데이터 탐색을 시작하세요.

⚡

GPU 가속

SkiaSharp로 구동되는 이미지 렌더링으로 빠르고 고품질 출력. 최신 천문 탐사의 대용량 FITS 파일에 최적화.
