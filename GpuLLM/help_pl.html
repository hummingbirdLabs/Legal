Co Potrafi GpuLLM

GpuLLM przekształca Twój komputer w kompletną lokalną stację roboczą AI.

Uruchamiaj LLM Lokalnie

Ładuj modele GGUF i rozmawiaj na GPU.

Monitoruj Sprzęt

GPU, VRAM, temperatura w czasie rzeczywistym.

Benchmarkuj Modele

Mierz TTFT, tokeny/s i dokładność MMLU/C-Eval.

Szybki Start

  1. Pobierz i zainstaluj

    Z Microsoft Store.

  2. Skonfiguruj token

    Utwórz token Read na huggingface.co/settings/tokens.

  3. Pobierz model

    Użyj filtra Mój Sprzęt.

  4. Załaduj model

    Kliknij Załaduj.

  5. Zacznij czatować

    Przejdź do Czat Benchmark.

Inferencja i Benchmark

Backend

CUDA 12 → Vulkan → CPU.

  • TTFT — Czas do pierwszego tokena.
  • Tokeny/s — Prędkość generowania.

Wskazówki

  • Zacznij od małych: Testuj modele 1-3B.
  • Sprawdź VRAM: Użyj filtra Mój Sprzęt.
  • Błąd 401/403: Potrzebny token HuggingFace.
