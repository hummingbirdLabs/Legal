ดูด้วยตัวคุณเอง

อินเทอร์เฟซที่ลื่นไหล ทันสมัย ทำให้การอนุมาน LLM แบบโลคัลและการตรวจสอบ GPU เป็นเรื่องง่าย

คลังโมเดล — เรียกดู 13 โมเดลที่คัดสรรในหมวด Coding, Chat และ Reasoning พร้อมการกรอง RAM ตามฮาร์ดแวร์
Chat Benchmark — แชทกับโมเดล เปรียบเทียบสองโมเดลแบบเคียงข้าง (Responder & Reviewer) และตรวจสอบเมตริก GPU แบบเรียลไทม์
Benchmark Suite — ประเมินความสามารถของโมเดลด้วย MMLU & C-Eval ดูความแม่นยำตามหมวดหมู่และการแจกแจงตามระดับความยาก
รายงาน LLM Benchmark — เมตริกประสิทธิภาพโดยละเอียด การประมาณต้นทุนโทเค็น และคำแนะนำฮาร์ดแวร์ส่วนบุคคล
Features

ทุกสิ่งที่คุณต้องการ — และฟรี

GpuLLM เป็นชุดเครื่องมือ Windows ฟรีที่สมบูรณ์แบบที่สุดสำหรับการประเมิน LLM แบบโลคัล นี่คือทุกสิ่งที่คุณได้รับ — ไม่มีค่าสมัครสมาชิก ไม่มีเพย์วอลล์

🧠

การอนุมาน LLM แบบโลคัล AI

รันโมเดลภาษาขนาดใหญ่รูปแบบ GGUF โดยตรงบน GPU ของคุณโดยใช้ LLamaSharp พร้อมแบ็กเอนด์ llama.cpp รองรับ CUDA, Vulkan และ CPU fallback — เลือกแบ็กเอนด์ที่ดีที่สุดโดยอัตโนมัติ

📂

นำเข้าโมเดล GGUF จากโลคัล AI

นอกเหนือจากแคตตาล็อกที่คัดสรรแล้ว นำเข้าโมเดลรูปแบบ GGUF จากดิสก์ของคุณเอง ชี้ GpuLLM ไปยังไฟล์ GGUF ของคุณและเริ่มแชทกับโมเดลที่กำหนดเองหรือชุมชนทันที — ไม่ต้องอัปโหลดคลาวด์

📊

การตรวจสอบ GPU แบบเรียลไทม์

ติดตามการใช้งาน GPU, การใช้ VRAM, อุณหภูมิ และการใช้พลังงานแบบเรียลไทม์ รวมกราฟ sparkline ย้อนหลัง 60 วินาที และรองรับหลาย GPU พร้อมการค้นพบอุปกรณ์อัตโนมัติ

⚡

Chat Benchmark และการประหยัดต้นทุน AI

แชทกับโมเดลใดๆ และวัดความเร็วการสร้างโทเค็นของ GPU (tokens/s) เวลาถึงโทเค็นแรก และความหน่วงรวมทันที ตัวประมาณต้นทุนในตัวเปรียบเทียบการอนุมานโลคัลกับราคา API คลาวด์ — แสดงจำนวนเงินที่คุณประหยัดได้จากการรันโมเดลแบบโลคัล

📦

คลังโมเดลและการจัดการ

เรียกดู 13 โมเดลที่คัดสรร รวมถึง Qwen2.5-Coder, DeepSeek-R1, Llama 3.2, Gemma 3 และ Mistral กรองตามหมวดหมู่ — Coding, Chat หรือ Reasoning — และตรวจสอบความเข้ากันได้ของ VRAM ในคลิกเดียว

🔽

ดาวน์โหลดจาก HuggingFace

ดาวน์โหลดโมเดลโดยตรงจาก HuggingFace Hub หรือ ModelScope รองรับการดาวน์โหลดต่อเนื่องพร้อมการตรวจสอบ SHA-256, การสำรองหลายแหล่ง และการจัดการความยินยอมของใบอนุญาต

💬

การสนทนาสองโมเดล: Responder และ Reviewer AI

บุกเบิกรูปแบบการโต้ตอบใหม่ — กำหนดสองบทบาท LLM ในการแชทเดียว: Responder ที่ตอบคำถาม และ Reviewer ที่ประเมินและปรับปรุงคำตอบอย่างมีวิจารณญาณ รูปแบบ "LLM-as-a-Judge" / "Agentic Debate" นี้ช่วยให้คุณใช้จุดแข็งของสองโมเดลพร้อมกันเพื่อการสนทนาที่มีคุณภาพสูงขึ้นและแก้ไขตัวเองได้

😺

GPU Spirit Animation

Prism Cat — มาสคอตเคลื่อนไหวสดที่ตอบสนองต่อระดับการใช้งาน GPU ของคุณ ดูสถานะอารมณ์และสีของแมวเปลี่ยนแบบเรียลไทม์เมื่อภาระงานการอนุมานเพิ่มขึ้น

🔌

การประเมินความสามารถของโมเดล AI

วัดประสิทธิภาพโมเดลของคุณกับชุดคำถามที่ครอบคลุม: คำถามภาษาอังกฤษ 100 ข้อ (MMLU สุ่มจาก 57 วิชา) และ คำถามภาษาจีน 100 ข้อ (C-Eval) ครอบคลุมมิติมากกว่า 10 มิติ ได้แก่ คณิตศาสตร์ ฟิสิกส์ เคมี ชีววิทยา วิทยาการคอมพิวเตอร์ ประวัติศาสตร์ วรรณกรรม เศรษฐศาสตร์ ปรัชญา และภูมิศาสตร์ รับความแม่นยำตามหมวดหมู่ การแจกแจงตามระดับความยาก และรายงาน Markdown ที่ส่งออกได้

📝

รายงานรายละเอียดการอนุมาน

การอนุมานแต่ละครั้งสร้างรายงานโดยละเอียดพร้อมสายโซ่แบ็กเอนด์ เมตริก GPU สูงสุด และสถิติระดับโทเค็น ส่งออกการสนทนาและรายงานทั้งหมดเป็นเอกสาร PDF A4
