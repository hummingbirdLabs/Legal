External API Services

HuggingFace Hub API

GpuLLM optionally communicates with the HuggingFace Hub public API (https://huggingface.co) to search and download AI model files at your explicit request. No data is sent to HuggingFace unless you initiate a model search or download. HuggingFace's own privacy practices are governed by their Privacy Policy.

ModelScope API

As an alternative model source, GpuLLM can connect to the ModelScope public API (https://modelscope.cn) to search and download AI model files at your explicit request. ModelScope's own privacy practices are governed by their policies available at https://modelscope.cn.

NVIDIA System Management Interface

NVIDIA SMI

GpuLLM uses the nvidia-smi command-line tool (installed with NVIDIA GPU drivers) to collect GPU metrics such as utilization, temperature, power draw, and VRAM usage. nvidia-smi is proprietary software provided by NVIDIA Corporation and is subject to the NVIDIA Software License Agreement. GpuLLM does not redistribute nvidia-smi; it invokes the tool only if already present on your system as part of your NVIDIA driver installation.

Curated Model License Notices

Important: GpuLLM provides a built-in library of curated model entries to help you discover and download AI models. GpuLLM does not redistribute these model files. Models are downloaded directly from their respective public repositories (HuggingFace Hub or ModelScope) at your explicit request.
