本地大模型推理 AI

使用 LLamaSharp 配合 llama.cpp 后端，直接在 GPU 上运行 GGUF 格式的大语言模型。支持 CUDA、Vulkan 及 CPU 回退——自动选择最优可用后端。

导入本地 GGUF 模型 AI

除了精选目录，还可从本地磁盘导入任何 GGUF 格式的模型。将 GpuLLM 指向你自己的 GGUF 文件，即可立即与自定义或社区模型对话——无需云端上传。

实时 GPU 监控

实时追踪 GPU 利用率、显存占用、温度和功耗。包含 60 秒迷你趋势图和自动设备发现的多 GPU 支持。

⚡

聊天评测与成本节省 AI

与任何模型对话，即时测量 GPU 的 Token 生成速度（tokens/s）、首 Token 延迟和总延迟。内置成本估算器对比本地推理与云端 API 定价——精确展示本地运行模型节省了多少费用。

模型库与管理器

浏览 13 个精选模型，包括 Qwen2.5-Coder、DeepSeek-R1、Llama 3.2、Gemma 3 和 Mistral。按类别筛选——编程、聊天或推理——一键检查显存兼容性。

🔽

HuggingFace 下载

直接从 HuggingFace Hub 或 ModelScope 下载模型。支持断点续传、SHA-256 校验、多源回退和许可协议管理。

双模型对话：回答者与评审者 AI

开创全新交互模式——在单次对话中分配两个大模型角色：一个回答者回答问题，一个评审者批判性地评估并改进答案。这种"大模型即裁判" / "智能体辩论"模式在 AI 研究界广受讨论，让你同时利用两个不同模型的优势，获得更高质量、自我纠错的对话体验。

棱镜猫——一只实时动画吉祥物，根据 GPU 利用率水平做出反应。观察猫咪的情绪状态和颜色随推理工作负载实时变化。

模型能力评估 AI

使用全面的题库评测你的模型：100 道英文题（MMLU，涵盖 57 个学科）和100 道中文题（C-Eval），覆盖 10+ 个维度，包括数学、物理、化学、生物、计算机科学、历史、文学、经济学、哲学和地理。获取各学科准确率、难度级别分布和可导出的 Markdown 报告。
