Бенчмаркинга локальных LLM

GpuLLM — бесплатное Windows-приложение для всесторонней оценки локальных LLM: измерение скорости инференса (токены/с, задержка) и способностей модели (MMLU, C-Eval). Всё работает 100% офлайн — чат, бенчмарк, мониторинг GPU, управление моделями — только загрузка моделей требует интернета. Найдите идеальную модель для вашего оборудования за минуты. Без облачной зависимости, без API-ключей, данные никогда не покидают ваш компьютер.
